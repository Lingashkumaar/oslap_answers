# experiment 6 answers

clone your repo
```
git clone <your_repo>
```
cd into your repo
```
cd <your_repo>
```
check status of the repo
```
git status
```
create a file 
```
touch test.txt
echo "hello" > test.txt
```
add the file in the repo
```
git add .
```
commit the changes 
```
git commit -m "added test.txt"
```
push the repo
```
git push origin main
```
